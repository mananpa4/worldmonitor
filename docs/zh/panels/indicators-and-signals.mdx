WorldMonitor 附有许多**磁贴式**指标、关联和信号面板 — 紧凑的面，每个显示一两个读数，设计为与深度分析面板组合使用。本页是集合参考，使每个磁贴可查找，而无需为每个面板写一页。

当磁贴为空时，请检查 [数据来源](/zh/data-sources) 和 [健康端点](/zh/health-endpoints) 了解上游状态。

## 共享行为

- 以上所有面板均**免费**，除非另有说明 — 在 `src/config/panels.ts` 中无 `premium: 'locked'`。
- 紧凑磁贴跨多个变体注册；按变体的注册决定磁贴是否默认启用。确认可用性时请直接检查 `src/config/panels.ts`。
- 大多数磁贴式面板从 bootstrap 载荷（`getHydratedData(...)`）水合，并回退到新的 RPC 调用，使初始绘制即时完成。
